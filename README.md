Запускать нужно файл scheduler.py, указав, storage_path, куда скачается архив с данными, и будут размещены файлы конфигурации.
Usage: python scheduler.py <storage_path>

При запуске программы сначала скачается архив с данными, затем распакуется. После распакуются архивы внутри него и конвертируются в rinex-файлы. 
Одновременно запустится планировщик, который будет скачивать архив каждый день в 23:00:00.
После скачивания и конвертации запустится orchestrator, который запустит стримеры для каждой скачанной станции. Для каждой станции каждые 30 секунд будут выводиться данные из файлов.
